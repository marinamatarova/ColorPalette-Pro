ColorPalette Pro — Генератор цветовых палитр на 7 языках
ColorPalette Pro — коллекция из семи независимых реализаций генератора гармоничных цветовых палитр. Каждая версия работает на своём языке программирования и предлагает мощный инструмент для дизайнеров, разработчиков и художников: создание палитр по различным цветовым схемам, визуализация, экспорт и работа с цветовыми моделями.

✨ Уникальные возможности (общие для всех версий)
🎯 Генерация гармоничных палитр по 5 схемам:

Монохромная — оттенки одного цвета.

Комплементарная — основной цвет + противоположный.

Аналоговая — соседние цвета на цветовом круге.

Триадная — 3 цвета через 120°.

Тетрадная — 4 цвета (две комплементарные пары).

🌈 Цветовые модели: HEX, RGB, HSL (ввод и вывод).

🖌️ Визуализация:

В CLI: цветные квадраты в терминале с использованием ANSI-кодов.

В GUI (Python, Java, C#): интерактивный просмотр палитры.

В вебе (JavaScript, PHP): живой предпросмотр с возможностью копирования.

💾 Экспорт палитры в форматы:

CSS (переменные)

JSON

HTML (таблица с цветами)

PNG (изображение палитры — в некоторых версиях)

📋 История созданных палитр с возможностью повторного использования.

🎲 Генерация случайного базового цвета или ввод своего.

🔢 Настройка количества цветов в палитре (от 2 до 10).

🖥️ Интерфейсы:

Командная строка (CLI) с богатыми опциями.

Интерактивный режим (ввод параметров).

Графический интерфейс (Python – Tkinter, Java – Swing, C# – WinForms).

Веб-интерфейс (JavaScript и PHP).
